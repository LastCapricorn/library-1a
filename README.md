# library-1a

The 3rd Project from [The Odin Project](https://www.theodinproject.com)'s Full Stack JavaScript Path.

I loved and hated this one in equal measure.

It took me literally three days to get the star rating running as desired. Afterwards I felt terribly stupid.

I now know that I can style an element using CSS, but Javascript doesn't do anything because the NodeList is empty - funny!

And scrollbars where they aren't supposed to be... annoying.

## But okay, here we go

- One library, but you can have several bookshelfs
  - which you can also remove

- Add new books to the current bookshelf
  - change the *read status* of a book
  - change the *star rating*
  - remove a book

- Sort books ascending or descending by title

- Change the color scheme

- The library uses ***localStorage***
  - clear *localStorage* before closing the page (see info button) or just for testing purposes

- Single bookcards to swipe on mobile devices

- Changes to *normal* CSS grid with larger screens

- At 700px width a button appears to change from *card view* to *table view*
  - falls back to *card view* when resizing to smaller width

***

The *toolbar buttons* seem to be a bit tiny I have to admit (at least on my mobile phone), but nowadays, we have genetically inherited touchscreen fingers - right?

Here is the [**Live Preview**](https://lastcapricorn.github.io/library-1a/), have fun!
